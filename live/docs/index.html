Quick Start

Phone Camera

  1. 01. Select PHONE in the source selector (default).
  2. 02. Tap CONNECT — the app requests camera permission on first use.
  3. 03. The stream starts automatically. Tap ⟳ FLIP to switch front/back camera.

USB Mirrorless Camera

  1. 01. Enable USB Streaming or UVC mode on your camera.
  2. 02. Connect the camera to your phone via USB-C OTG cable.
  3. 03. Select USB CAMERA and tap CONNECT.
  4. 04. Confirm the USB permission dialog, then choose the desired resolution.
Pro Tip

After connecting, tap the QR button next to the URL to generate a QR code — anyone on the same Wi-Fi can scan it to open the web monitor instantly.

Camera Sources

📱

Phone Camera (PHONE)

Uses CameraX ImageAnalysis pipeline. Supports front and rear cameras. Resolution: 480p / 720p / 1080p (configurable). Tap FLIP anytime to switch without disconnecting.

🎥

USB Mirrorless (USB CAMERA)

UVC via USB Host. Compatible with Sony A7 series, Canon R series, and Fujifilm (with USB Streaming / UVC mode enabled). Requires USB-C OTG cable. Resolution selected after connecting.

Monitor Setup

📺

Android TV

Install the Infinity Live companion app on Android TV. It discovers the phone automatically via Wi-Fi NSD. No IP entry needed.

📡

Chromecast / Mi Stick

Tap CAST in the app and select your device. The receiver uses chain-loading for ~15–25fps. PNA headers ensure Mi Stick compatibility.

🌐

Web Browser

Open http://IP:8080 on any device in the same network. Supports MJPEG (~25fps) with automatic fallback to snapshot polling for Safari.

Settings

Tap the ⚙ button to open the settings menu. Each item shows its current value:

Web Rotation 0° / 90° / 180° / 270° — affects the browser web monitor
TV Rotation 0° / 90° / 180° / 270° — affects the Android TV app
Cast Rotation 0° / 90° / 180° / 270° — affects Chromecast / Mi Stick
JPEG Quality 60 / 70 / 85 (default) / 95 — applies to all outputs
Cast Quality 60 (default) / 70 / 85 / 95 — independent Cast quality
Camera Resolution 480p / 720p / 1080p — phone camera only
Watermark Select PNG/JPEG from gallery — shown bottom-right on all outputs, persisted between sessions

Web Monitor

When connected, the address http://IP:8080 is shown in the app with:

  • ✔ Tap the URL to copy it to clipboard.
  • ✔ Tap QR to generate a QR code (512×512) scannable by other devices on the network.
  • ✔ The web page includes an fps counter, rotation button for the web output, fullscreen mode, and automatic Safari fallback.
  • ✔ TV and Cast rotation: controlled exclusively in the operator app.
