üöÄ Quick Start Guide

Step 1: Install

Does it work with RAW files?

Yes. We recommend converting your RAW layer to a Smart Object before processing for best results.

What is 16-bit mode?

It's a high-depth mode that prevents "banding" (visible lines) in background gradients. Our plugin uses this internally.
