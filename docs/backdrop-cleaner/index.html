üöÄ Quick Start Guide

Get started with Infinity Backdrop Cleaner in 5 simple steps:

Step 1: Install & Activate

Download from Creative Cloud, install, and activate with your license key.

Step 2: Open Your Image

Open a photo with an infinite backdrop in Photoshop (works best with white/gray seamless backgrounds).

Step 3: Prepare for Painting

Click "Prepare for Painting" button in the plugin panel. This creates a mask and configures your brush.

Step 4: Paint Over Imperfections

Hold Alt (Option on Mac) and click on a clean background area to sample color. Then paint over shadows, wrinkles, or imperfections.

Step 5: Process Image

Click "Process Image" to generate the seamless backdrop. Use eraser tool on "Original Background" layer to recover subject details if needed.

üí° Pro Tip: Don't worry about painting over your subject! The plugin preserves the original in a separate layer, so you can easily recover any details using the eraser tool.

üé® How to Use

Detailed Workflow

1. Prepare Your Image

For best results, your image should have:

  • A white, gray, or light-colored seamless backdrop
  • Subject clearly separated from background
  • Good lighting (studio setup works best)

2. Mask & Prepare Phase

Click "Prepare for Painting". The plugin will:

  • Automatically detect your subject using AI
  • Create a protective mask
  • Configure brush settings optimally
Advanced Controls: Click "Show Advanced Controls" to adjust mask expansion and feathering if the automatic detection isn't perfect.

3. Paint Over Imperfections

With the painting layer active:

  1. Hold Alt (Option on Mac)
  2. Click on a clean background area to sample the color
  3. Release Alt and paint over shadows, wrinkles, or marks
  4. Paint generously - don't worry about covering the subject
üí° Pro Tip: Use a large, soft brush (200-500px) for faster coverage. The plugin preserves your subject, so paint freely!

4. Process & Finalize

Click "Process Image". The plugin will:

  • Analyze your painting
  • Generate seamless backdrop
  • Preserve original in "Original Background" layer
  • Apply professional blur and noise reduction

5. Refine (Optional)

If any subject details were affected:

  1. Select the Eraser tool
  2. Erase on the "Original Background" layer
  3. This reveals the clean backdrop underneath

‚öôÔ∏è Advanced Features

Blur Types

The plugin offers two blur algorithms:

  • Gaussian Blur: Smoother, more uniform blur. Best for most cases.
  • Surface Blur: Edge-preserving blur. Use when you want to keep some texture detail.

Advanced Controls

Click "Show Advanced Controls" in Step 2 to access:

  • Expand: Enlarges the cleaned area
  • Feather: Softens edges for smoother blending
  • Blur Radius: Controls blur intensity
  • Noise: Adds subtle grain to prevent banding (especially useful for prints)

16-Bit Processing

The plugin automatically processes in 16-bit mode to preserve maximum quality. This prevents color banding and ensures smooth gradients, especially important for:

  • RAW files
  • High-end prints
  • Professional retouching

‚ùì Frequently Asked Questions

Can I use it with RAW files?

Yes! Convert your RAW layer to a Smart Object first, then use the plugin. This maintains non-destructive editing capabilities.

Does it work with dark backgrounds?

The plugin is optimized for light-colored infinite backdrops (white, gray, cream). Dark backgrounds may require more manual refinement.

Why isn't my subject detected correctly?

Try these solutions:

  • Ensure sufficient contrast between subject and background
  • Manually adjust the mask using "Advanced Controls"
  • Use Photoshop's Quick Selection tool to refine the mask first

Can I process multiple images at once?

Currently, the plugin processes one image at a time. We recommend using Photoshop Actions to batch process if needed.

üîß Troubleshooting

Plugin doesn't appear in Photoshop

Solution:

  1. Go to: Window ‚Üí Extensions ‚Üí Infinity Backdrop Cleaner
  2. If not listed, restart Photoshop
  3. Still missing? Reinstall from Creative Cloud Desktop

License activation fails

Possible causes:

  • No internet connection: Check your connection
  • Incorrect key: Copy/paste the key from the email (don't type manually)
  • Already activated twice: Deactivate an old device first

Processing is very slow

Solutions:

  • Large images (>50MP) can take 30-60 seconds - this is normal
  • For faster preview, resize image before processing
  • Close other applications to free up RAM
  • Ensure Photoshop has adequate scratch disk space

"Could not detect subject" error

Solutions:

  • Create a manual selection with Quick Selection tool first
  • Ensure your image has sufficient contrast
  • Convert layer to Smart Object if working with RAW
