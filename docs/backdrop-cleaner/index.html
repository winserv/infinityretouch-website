üöÄ Quick Start Guide

Achieve seamless studio backdrops in seconds with this professional workflow:

1

Installation

Install via Creative Cloud or use the Manual Method for direct installation.

2

Preparation

Open your image (16-bit recommended). Click 'Prepare for Painting'. The AI Engine will automatically mask your subject.

3

Sampling

Hold Alt/Option and click on a clean area of the backdrop to sample the target tone.

4

Painting

Paint generously over wrinkles, shadows, and footprints. You can paint over the subject's edges; the AI mask protects them.

5

Processing

Click 'Process Image'. The plugin creates a new layer with a perfectly blended, band-free gradient.

üí° Pro Tip: Edge Recovery

If the AI mask hides fine details (like flyaway hair), simply use a black brush on the 'Mask' layer or the Eraser tool on the 'Original Background' layer to reveal them instantly.

üõ†Ô∏è Manual Installation (Alternative)

Use this direct deployment method if you experience sync issues with Creative Cloud:

1

Extract Files

Rename the installer from .ccx to .zip. Extract the contents into a folder named 'InfinityRetouch'.

2

Locate Plugin Folder

Navigate to the Photoshop UXP directory:
Win: C:\Program Files\Common Files\Adobe\UXP\extensions
Mac: ~/Library/Application Support/Adobe/UXP/Plugins

3

Deploy

Move the 'InfinityRetouch' folder directly into the extensions (or Plugins) folder.

4

Launch

Open Photoshop and navigate to Window ‚Üí Plugins ‚Üí Infinity Backdrop Cleaner.

!

Mac Users: The '~' symbol refers to your User Home folder. If 'Library' is hidden, open Finder, hold 'Option', click 'Go', and select 'Library'.
