🚀 Guia Rápido

Obtenha fundos de estúdio perfeitos em segundos seguindo este fluxo profissional:

1

Instalação

Instale via Creative Cloud ou use o Método Manual para instalação direta.

2

Preparação

Abra sua imagem (16-bits recomendado). Clique em 'Preparar para Pintura'. A IA isolará o assunto automaticamente.

3

Amostragem

Segure Alt/Option e clique em uma área limpa do fundo para capturar o tom alvo.

4

Pintura

Pinte generosamente sobre rugas, sombras e pegadas. Você pode pintar sobre as bordas do modelo; a máscara de IA as protege.

5

Processamento

Clique em 'Processar Imagem'. O plugin cria uma nova camada com um gradiente suave e livre de banding.

💡 Dica: Recuperação de Detalhes

Se a máscara de IA ocultar detalhes finos (como cabelos soltos), basta usar um pincel preto na camada 'Mask' ou a Borracha na camada 'Original Background' para revelá-los.

💿 Instalação Padrão

Requisitos do Sistema

  • ✔ Adobe® Photoshop® 2022+ (v23.0 ou superior)
  • ✔ Windows 10/11 (64-bit) ou macOS 11+ (Big Sur, Monterey, Ventura, Sonoma)
  • ✔ Suporte Nativo para Apple Silicon (Série M) e Intel
  • ✔ Mínimo de 8GB RAM (16GB Recomendado para arquivos grandes)

Método Padrão (Creative Cloud)

  1. 01. Acesse a página do Adobe Exchange ou clique duas vezes no arquivo .ccx.
  2. 02. O aplicativo Creative Cloud Desktop abrirá. Clique em 'Instalar'.
  3. 03. Aguarde a notificação de confirmação de instalação.
  4. 04. Reinicie o Photoshop para inicializar o motor UXP.

🛠️ Instalação Manual (Alternativa)

Use este método de implantação direta caso enfrente problemas de sincronização com a Creative Cloud:

1

Extrair Arquivos

Renomeie o instalador de .ccx para .zip. Extraia o conteúdo para uma pasta chamada 'InfinityRetouch'.

2

Localizar Pasta UXP

Navegue até o diretório UXP do Photoshop:
Win: C:\Arquivos de Programas\Common Files\Adobe\UXP\extensions
Mac: ~/Library/Application Support/Adobe/UXP/Plugins

3

Implantar

Mova a pasta 'InfinityRetouch' diretamente para dentro da pasta extensions (ou Plugins).

4

Iniciar

Abra o Photoshop e vá para Janela → Plugins → Infinity Backdrop Cleaner.

!

Usuários Mac: O símbolo '~' refere-se à sua pasta de Usuário. Se a 'Biblioteca' estiver oculta, abra o Finder, segure 'Option', clique em 'Ir' e selecione 'Biblioteca'.

🎨 Detalhes do Fluxo

Motor de Alta Profundidade

O plugin converte o processamento para 16-bits internamente. Isso garante gradientes suaves e previne artefatos de 'banding' comuns em fotografia de estúdio.

Máscara Inteligente AI

Utilizamos os modelos mais recentes de reconhecimento de sujeito da Adobe. Ele isola modelos com precisão, mesmo com roupas complexas ou adereços.

Separação de Frequência

Durante o processamento, o motor separa a textura da cor, garantindo que suas pinceladas se misturem naturalmente com o grão original do piso.

⚙️ Recursos Avançados

Seleção de Algoritmo

Gaussiano: Suavidade máxima. Ideal para fundos de papel contínuo ou cicloramas limpos.

Superfície: Desfoque inteligente que preserva textura. Melhor para pisos de concreto, madeira ou texturizados.

Tecnologia Anti-Banding

O banding digital pode arruinar uma impressão. Nosso motor aplica um padrão matemático de dithering durante a fusão final, garantindo arquivos prontos para impressão profissional.

❓ Perguntas Frequentes

Posso instalar no meu Desktop e no Notebook?

Sim. Cada licença inclui 2 ativações simultâneas.

Aparece o erro 'Plugin não compatível'.

Por favor, atualize o Photoshop para a versão 23.0 (2022) ou mais recente. Versões antigas não possuem a arquitetura UXP necessária.

Funciona em fundos coloridos?

Sim, funciona em qualquer cor sólida. Para melhores resultados, garanta que a iluminação esteja relativamente uniforme.
