Política de Privacidade

Última atualização: Janeiro 2026

A Infinity Retouch ("nós"), um produto da WINSERV TECNOLOGIA DA INFORMACAO, está comprometida em proteger sua privacidade e fornecer transparência sobre como lidamos com seus dados.

1. Coleta de Dados

Coletamos o mínimo absoluto de dados necessários para o funcionamento do software e para fornecer suporte:

  • Dados Pessoais: Seu Nome e endereço de Email, fornecidos durante o checkout.
  • Dados de Licença: Uma Impressão Digital da Máquina (hash de hardware anonimizado) para validação de licença e para gerenciar seu limite de 2 dispositivos via nosso provedor, Keygen.sh.
  • Dados de Pagamento: Processados com segurança por nosso comerciante de registro, Kiwify. Não armazenamos ou temos acesso aos detalhes do seu cartão de crédito.

2. Uso dos Dados

Seus dados são usados exclusivamente para os seguintes propósitos:

  • Para entregar sua chave de licença única via email.
  • Para validar a ativação do seu software e prevenir pirataria.
  • Para fornecer suporte ao cliente quando solicitado.

Motor de Processamento Local

Importante: Infinity Retouch é um plugin local. Suas fotos nunca saem do seu computador. Não fazemos upload, armazenamos ou temos acesso a nenhuma das imagens que você processa, garantindo total segurança para o trabalho dos seus clientes.

3. Serviços de Terceiros

Compartilhamos dados apenas com provedores de serviços essenciais e seguros que são críticos para nossas operações:

  • Keygen.sh (Servidor de Licenciamento)
  • Kiwify (Processamento de Pagamento)
  • Make.com (Automação para entrega de email)

4. Entre em Contato

Se você tiver alguma dúvida sobre esta Política de Privacidade, entre em contato conosco em sales@infinityretouch.com.
