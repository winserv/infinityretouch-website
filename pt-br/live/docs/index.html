Início Rápido

Câmera do Celular

  1. 01. Selecione PHONE no seletor de fonte (padrão).
  2. 02. Toque em CONNECT — o app solicita permissão de câmera na primeira vez.
  3. 03. O stream inicia automaticamente. Toque em ⟳ FLIP para alternar câmera frontal/traseira.

Câmera Mirrorless via USB

  1. 01. Ative o modo USB Streaming ou UVC na câmera.
  2. 02. Conecte a câmera ao celular via cabo USB-C OTG.
  3. 03. Selecione USB CAMERA e toque em CONNECT.
  4. 04. Confirme o dialog de permissão USB e escolha a resolução desejada.
Dica Pro

Após conectar, toque no botão QR ao lado da URL para gerar um QR code — qualquer pessoa na mesma rede pode escanear e abrir o monitor instantaneamente.

Fontes de Câmera

📱

Câmera do Celular (PHONE)

Usa pipeline CameraX ImageAnalysis. Suporta câmeras frontal e traseira. Resolução: 480p / 720p / 1080p (configurável). Toque em FLIP para alternar sem desconectar.

🎥

Mirrorless USB (USB CAMERA)

UVC via USB Host. Compatível com Sony A7, Canon R e Fujifilm (com modo USB Streaming / UVC ativado). Requer cabo USB-C OTG. Resolução selecionada após conectar.

Configuração de Monitores

📺

Android TV

Instale o app companion Infinity Live no Android TV. Ele descobre o celular automaticamente via Wi-Fi NSD. Sem necessidade de digitar IP.

📡

Chromecast / Mi Stick

Toque em CAST no app e selecione o dispositivo. O receiver usa chain-loading para ~15–25fps. Headers PNA garantem compatibilidade com Mi Stick.

🌐

Navegador Web

Abra http://IP:8080 em qualquer dispositivo na mesma rede. Suporta MJPEG (~25fps) com fallback automático para polling no Safari.

Configurações

Toque no botão ⚙ para abrir o menu de configurações. Cada item exibe o valor atual:

Rotação Web 0° / 90° / 180° / 270° — afeta o monitor do navegador
Rotação TV 0° / 90° / 180° / 270° — afeta o app Android TV
Rotação Cast 0° / 90° / 180° / 270° — afeta Chromecast / Mi Stick
Qualidade JPEG 60 / 70 / 85 (padrão) / 95 — aplicada a todas as saídas
Qualidade Cast 60 (padrão) / 70 / 85 / 95 — qualidade independente para Cast
Resolução Câmera 480p / 720p / 1080p — somente câmera do celular
Marca d'água Selecione PNG/JPEG da galeria — aparece canto inferior direito em todas as saídas, persiste entre sessões

Web Monitor

Quando conectado, o endereço http://IP:8080 é exibido no app com:

  • ✔ Toque na URL para copiar para a área de transferência.
  • ✔ Toque em QR para gerar um QR code (512×512) escaneável por outros dispositivos na rede.
  • ✔ A página inclui contador de fps, botão de rotação para a saída web, modo fullscreen e fallback automático para Safari.
  • ✔ Rotação TV e Cast: controladas exclusivamente no app do operador.
