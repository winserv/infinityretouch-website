Quick Start

1

Install

Download Infinity Theater from Google Play and install it on your Android phone.

2

Start Session

Open the app and tap START SESSION. The server starts and your phone's IP address is displayed.

3

Configure Camera

In your camera's FTP settings, enter the IP shown, port 2121, user infinity, password theater, passive mode.

4

Connect Monitors

Open http://IP:8080 in any browser, cast to Chromecast, or install the TV app on Android TV. Photos appear the moment the camera uploads them.

Pro Tip

Set your camera's FTP transfer to "Each Shot" (not batch) for the lowest possible latency on screen.

FTP Configuration

Use exactly these settings in your camera's FTP transfer menu:

Server / Host IP shown in the app (e.g. 192.168.1.15)
Port 2121
Username infinity
Password theater
Mode Passive (PASV)
Destination Folder / (root) or leave blank

Compatible with Sony (Wi-Fi FTP), Canon (FTP Transfer), Nikon (WT series), Fujifilm (Wireless Settings > FTP), and any camera with FTP transfer support.

Monitor Setup

🌐

Web Browser

Open http://IP:8080 in any browser on the same Wi-Fi network. Works on laptops, tablets, phones and most Smart TV browsers.

📡

Chromecast

Tap the Cast icon at the top of the app and select your Chromecast or TV with built-in Chromecast.

📺

Android TV

Install the Infinity Theater companion app on your Android TV. It auto-discovers the phone via Wi-Fi — no IP entry needed.

🖥️

Samsung / LG Smart TV (DLNA)

Tap the DLNA pill in the app, select your TV from the list, and photos are pushed directly. No app installation needed on the TV.

Display Controls

Tap the gear icon in the center of the screen to open the settings menu. Each output has independent rotation:

  • ✔ Chromecast: Portrait / Inverted
  • ✔ Web Monitor / Android TV: 0° / 90° / 180° / 270°
  • ✔ DLNA: independent rotation in the DLNA section

Rotation settings are saved per output and persist across sessions.

Production Tips

Quality Settings

Choose Full HD (1920px) for maximum speed, 2K (2560px) for balance, or 4K (3840px) for maximum quality on large TVs.

Disk Cache

Photos are cached to disk, allowing 100+ photo history without consuming device RAM. History is cleared when starting a new session.

Post-Session Review

Tap STOP SESSION to end shooting. The photo strip remains on screen for client review. Cleared only when starting a new session.

Burst Protection

When the camera shoots rapidly, the app prioritizes the latest photo and drops intermediates to prevent display lag.
